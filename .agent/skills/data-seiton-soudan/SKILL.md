# data-seiton-soudan (相談データ整頓スキル)

高校における相談者（生徒・保護者等）のデータを、その後の分析（傾向把握やレポート作成）にスムーズに活用できるよう、一定の形式に整頓するためのスキルです。

## このスキルを使用する時
- 散らばっている相談記録を一つの表（CSV/Excel形式など）にまとめたい時
- 表記の揺れ（記述の統一）を修正したい時
- 相談内容をカテゴリ分け（ラベル付け）して、定量的な分析ができる状態にしたい時
- 生徒名などの個人情報を匿名化・ID化したい時

## このスキルを使用しない時
- 相談業務のリアルタイムな入力
- 統計ソフト（RやPythonの分析ライブラリ）を用いた複雑な多変量解析の実行

## ワークフロー

### 1. データ構造の把握
現在のデータの形式（紙のメモ、Excel、テキストファイル等）と、最終的にどのような項目（列）が必要かを確認します。

### 2. データのクリーニング
- 重複データの削除
- 表記の統一（例：「保護者面談」と「親面談」を「保護者面談」に統一）
- 欠損値の扱い（空欄をどう埋めるか）の決定

### 3. 匿名化・安全性の確保
- 個人名や学籍番号を、分析用のランダムなIDに置き換えます。
- 住所、電話番号などの機密情報を削除または加工します。

### 4. カテゴリ化（ラベリング）
- 自由記述の相談内容から、キーワードを抽出したり、目的別のカテゴリ（「進路」「人間関係」「学業」など）を付与したりします。

## ヒアリング項目
整頓を開始する前に以下を確認してください：
- **現在のデータ形式**: どこに、どのような形で保存されていますか？
- **最終的な出力形式**: どのような項目（列名）を持ったデータにしたいですか？
- **匿名化の必要性**: 氏名を伏せる必要がありますか？それともIDで紐付ける必要がありますか？
- **カテゴリの定義**: どのような分類（タグ）を使いたいですか？
