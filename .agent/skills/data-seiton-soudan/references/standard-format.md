# 標準的な相談データフォーマット

高校での分析に使いやすい、標準的な列（カラム）の定義例です。

| 列名 (英語/ID) | 列名 (日本語) | 内容例 | 型 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| `date` | 相談日 | 2024/05/10 | 日付 | yyyy/mm/dd 形式を推奨 |
| `student_id` | 生徒ID | S12345 | 文字列 | 匿名化後のIDを推奨 |
| `grade` | 学年 | 1 | 数値 | |
| `class` | クラス | A | 文字列 | |
| `category` | 相談カテゴリ | 進路 | カテゴリ | 選択肢を固定すると分析しやすい |
| `summary` | 相談概要 | 志望校についての相談 | テキスト | |
| `duration` | 所要時間(分) | 30 | 数値 | |
| `counselor` | 担当者名 | 山田 | 文字列 | |

## 推奨されるカテゴリ例
- 進路（大学/専門/就職）
- 相談（人間関係/友人/家族）
- 学業（成績/授業/テスト）
- 生活（遅刻/欠席/体調）
- その他
