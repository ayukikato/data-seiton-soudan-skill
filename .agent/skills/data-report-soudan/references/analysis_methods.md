# 相談データ分析手法（高校募集戦略）

相談者データを最大限に活用し、次年度の生徒募集に繋げるための分析手法をいくつか提案します。

## 1. 地域別ターゲット分析
相談者の住所（郵便番号や区・市町村）を元に、どのエリアの志望者が多いか、あるいは今後アプローチを強めるべきエリアを特定します。

- **指標**: 相談者数（エリア別シェア）、特定エリアでの前年比伸び率
- **示唆**: どのエリアへの広報活動（ポスティング、中学校訪問等）を重点化すべきか。

## 2. 志望タイミング分析
いつ最初の相談が起きたか（学期、月、特定のイベント）と、その後の継続的な相談頻度を分析します。

- **指標**: 時系列相談件数、初めて相談に訪れたタイミング
- **示唆**: 併願から単願に変わったタイミングや、進路検討が本格化するピークを特定し、その時期に合わせたメッセージを発信。

## 3. 中学校との連携分析
どの中学校の生徒が相談に来ているかを整理します。

- **指標**: 中学校別の相談者数
- **示唆**: 学校説明会の開催校選定の優先順位付けや、特定の塾との連携強化の検討。

## 4. 興味関心と併願校の相関分析
相談者が興味を持っているコースや部活動と、比較されている他校（併願校）の関係を分析します。

- **指標**: 併願校、キーワード（部活動名など）の集計
- **示唆**: 本校独自の強みはどこにあるのか、あるいはどの分野で競合他校に流れているのかを明らかにする。

## レポートの表現案（表出方法）
- **ヒートマップ**: 地域別の相談密度を視覚的に表示
- **サンバースト図 / 円グラフ**: 興味関心の構成比
- **ファネル分析**: 資料請求 → イベント参加 → 個別相談 → 入学試験 → 入学 という段階ごとの推移
