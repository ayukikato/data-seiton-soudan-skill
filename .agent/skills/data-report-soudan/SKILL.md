# Data Report Soudan (相談データ分析・レポート作成)

相談者情報（名寄せ・整頓済み）を元に、次年度の生徒募集を成功させるための多角的な分析と戦略立案を支援します。

## このスキルを使用する時

- 蓄積された相談データを元に、募集戦略のヒントを得たい時
- 定期的な募集状況のレポートを作成したい時
- ターゲット層や注力すべきエリアを特定したい時
- 効果的な広報・イベントのタイミングを検討したい時

## このスキルを使用しない時

- データの形式を整えるだけの時（`data-seiton-soudan` または `data-seiton2-soudan` を使用）
- 相談内容を1件ずつ確認したいだけの時

## ワークフロー / 対応タスク

### 1. データの可視化とクロス集計
以下のような切り口でデータを集計し、現状を把握します。
- **地域別**: どのエリアからの相談が多いか（通学圏の確認）
- **中学校別**: どの中学校からのアプローチが強いか、または開拓の余地があるか
- **時期別**: 相談が集中する時期はいつか（広報戦略の評価）
- **学年別**: 進路決定のどの段階での相談が多いか
- **興味関心別**: どのコースや部活動への関心が強いか

### 2. トレンド分析と課題特定
- 前年度との比較を通じた傾向の変化
- 特定の属性（例：併願校など）と成約率（入学意向）の相関分析

### 3. レポート作成（表出方法）
分析結果を元に、以下のような構成でレポートを作成します。
- **エグゼクティブ・サマリー**: 主要な発見と結論（1枚でわかる概要）
- **詳細分析**: グラフや表を用いた具体的データ
- **改善提案・戦略**: 次年度の生徒募集活動への具体的なアクションプラン

## ヒアリング項目
レポート作成前に、以下の点を確認することがあります：
- **分析対象の期間**: どの期間のデータに焦点を当てますか？
- **比較対象**: 前年度との比較は必要ですか？
- **重視する指標 (KPI)**: 相談数、特定のエリアのシェア、第一志望率など、何が最も重要ですか？
- **レポートの相手**: 誰に向けたレポートですか？（内部共有、経営判断用など）
