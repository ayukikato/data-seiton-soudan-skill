# 受験番号Excel分析スキル

受験番号を共通のキー（ID）として、異なるExcelファイル（例：出願データ、試験結果、内申点データなど）を紐付け、多角的な分析を支援します。

## このスキルを使用する時
- 複数のExcelシートに分散している情報を「受験番号」で一つにまとめたい時
- 特定の受験番号に紐づく詳細な情報（住所、出身校、得点、判定など）を横断的に分析したい時
- 受験番号別の集計や統計（出身校別合格率、特待生比率など）を出したい時

## このスキルを使用しない時
- 受験番号が含まれていないデータの分析
- ファイルが一つしかなく、VLOOKUP等の単純な操作で完結する時
- 個人情報の厳格な分離が必要で、データの結合が禁止されている時

## ワークフロー / 対応タスク

### 1. データ構造の把握
- 各Excelファイルの列名を確認し、紐付けのキーとなる「受験番号」の形式（数値/文字列、ハイフンの有無など）を統一します。
- 分析に必要な項目（属性データ、得点データ、フラグ等）を特定します。

### 2. データ結合（マージ）
- 受験番号をキーとして、主となるテーブルに他の情報を結合します。
- 紐付け漏れ（片方のファイルにしかない番号）のチェックと処理を行います。

### 3. 多角的な分析
- 出身校、入試区分、得点帯などの軸でクロス集計を行います。
- 記述統計や相関関係（内申点と当日点の関係など）の算出を行います。

## ヒアリング項目
分析を開始する前に、以下の点を確認してください：
- キーとなる受験番号の列名は正確に何か（例：「受番」「受験No」など）
- 主軸となるファイルはどれか
- 最終的にどのようなアウトプット（表、グラフ、特定のリストなど）を求めているか

## 出力形式
- 結合後のデータセット（CSVまたは分析用データフレーム構造）
- 定量的な分析レポート（Markdown形式の表、要約統計量）
- 異常値や不整合データのリスト
