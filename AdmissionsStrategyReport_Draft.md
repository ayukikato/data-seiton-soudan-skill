# 2026年度 生徒募集戦略レポート（ドラフト）

## 1. 概要（Executive Summary）
本年度の相談データ（2,076件）を分析した結果、全体の約88%が併願層であり、単願（第一志望）層の拡大が次年度の最重要課題であることが浮き彫りになりました。

- **総相談数**: 2,076件（ユニーク相談者数）
- **単願率**: 12.0%（併願：1,827件 / 単願：249件）
- **主要な発見**:
    - **特定エリアの強み**: さいたま市桜区、富士見市において単願率（本気度）が14〜15%と高く、これらの地域が「熱い」ターゲットである。
    - **コース別の特性**: `STEAM` コースが圧倒的な単願率（42%）を誇り、本校の独自性を象徴する強力なフックとなっている。
- **結論**: 次年度は「数（相談数）」の維持に加え、「質（単願への転換）」を重視し、高単願率エリアへの重点投資と、STEAM教育を軸にしたブランディングを強化すべきである。

### 2.2 コース別志望傾向
コースによって募集の性質が大きく異なります。

- **S-uni (1,597件 / 単願率 12.1%)**: 本校の基盤。併願層が多いため、他校との差別化要因を明確にする必要がある。
- **STEAM (33件 / 単願率 42.4%)**: 募集数は少ないが、第一志望で狙ってくる層が極めて多い。新時代の看板として広報の主役に据えるべき。
- **E-uni (344件 / 単願率 9.3%)**: 上位層を狙うコースだが、併願率が最も高い。併願辞退を防ぐための入学後メリットの提示が課題。

## 4. 次年度に向けた具体的な戦略提案

### 4.1 広報戦略： 「STEAMの熱を全コースへ」
- STEAMコースの活動内容（探究学習、外部連携）を全コースの共通体験として広報し、「この学校に入れば新しい学びができる」というブランドを確立する。
- 単願率の高い「富士見市・志木市北部・さいたま市桜区」にフォーカスしたポスティングおよび塾訪問を、9月〜10月に集中実施。

### 4.2 イベント運営： 「併願を単願に変える体験設計」
- 個別相談において、現在の内申・偏差値の提示だけでなく、「本校でしかできない3年間のストーリー」を提示する個別提案型の相談会を実施。
- 所沢・朝霞エリア等の「併願多数エリア」向けの限定イベント（部活動体験、プレ入試等）を開催し、接触回数を増やす。

### 4.3 募集基準の最適化
- STEAMコースや、さいたま市桜区等の特定エリアからの単願希望者に対し、相談基準の柔軟な適用や、早期特典等のインセンティブを検討。

## 5. 結び
2026年度、本校は「選ばれる理由」が明確な学校へと進化する必要があります。
データの示す「エリア特性」と「コース特性」を最大限に活かし、効率的かつ熱量の高い募集活動を展開しましょう。
